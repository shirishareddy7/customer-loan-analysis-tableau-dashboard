# 📊 Customer Loan Analysis – Interactive Dashboard (Tableau)

## 🧾 Project Overview
This project presents a Tableau dashboard analyzing customer loan data from 2007–2011. 
The same dataset has been used across Excel, Power BI, and Tableau to demonstrate cross-tool data analysis and visualization capabilities.

## 🎯 Purpose
To analyze loan data and build an interactive dashboard that highlights trends, customer behavior, and risk patterns using Tablea

## 📂 Dataset Used
- Loan dataset with customer details, loan amount, interest rate, grade, and payment status  
- Time Period: 2007 – 2011  
- Dataset reused across Excel, Power BI, and Tableau dashboards  

## 📊 Questions (KPIs)
- Year-wise total loan amount trend  
- Revolving balance by loan grade and sub-grade  
- Verified vs non-verified payment comparison  
- State-wise loan amount by loan status  
- Home ownership vs last payment trends  


## ⚙️ Project Process
- Cleaned and prepared dataset  
- Performed data analysis  
- Created calculated fields and aggregations  
- Built interactive charts and dashboards  
- Applied filters and slicers for dynamic analysis  


## 📌 Key Insights
- Loan amount increased steadily over the years  
- Grade B and C loans contribute more to total balance  
- Verified customers show better repayment performance  
- Loan activity varies across states and months  
- Home ownership impacts repayment behavior  


## ✅ Final Conclusion
This project demonstrates how the same dataset can be analyzed using different tools. 
Tableau provides powerful interactive visualizations that enhance understanding of loan trends and support better decision-making.
